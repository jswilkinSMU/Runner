# Runner
A 2.5D platformer that is based off of the Run series on CoolmathGames

![Runner Banner]((https://github.com/jswilkinSMU/Runner/blob/main/RunnerHeroImage.png))

How to use:

  	 - Controls:
		    - Spacebar - Jump
		    - A - Turn Left
		    - D - Turn Right
		    - P - Pause
		    - R - Reset
